# golang-youtube-downloader

A Go port of [YoutubeDownloader](https://github.com/Tyrrrz/YoutubeDownloader) by Tyrrrz.

## About

This project is a Go implementation of YoutubeDownloader, a cross-platform application for downloading media from YouTube.

### Planned Features

- Download individual videos by URL
- Download entire playlists and channel content
- Search-based video discovery
- Multiple quality and format options
- Subtitle/caption support
- Metadata embedding
